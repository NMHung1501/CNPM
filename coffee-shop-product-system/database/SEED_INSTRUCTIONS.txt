Cách import dữ liệu mẫu (SQL Server)

1) Mở SQL Server Management Studio (SSMS)
2) Tạo database: DB_NAME (trong file .env) - ví dụ: DB_Coffee
3) Import schema:
   - Mở file: database\schema_mssql.sql
   - Execute

4) Import seed dữ liệu mẫu:
   - Mở file: database\seed_mssql.sql
   - Execute

Sau đó chạy server:
- http://localhost:5000/
- http://localhost:5000/admin.html
